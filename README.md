# MeFEEdu

> An Educational Mechanical Finite Element solver written in Rust


## Summary
A small program to solve 2D Beam problems, using the Direct Stiffness Matrix.

## Usage
To run this project install rust and Cargo, you will also need a working OpenGL installation for raylib to work.

To get help on usage call: `cargo run -- --help`

If you want to try the interactive mode from our Presentation use the `interactive`-Branch.
